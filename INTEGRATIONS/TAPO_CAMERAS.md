# TAPO CAMERAS  

## Install
Install [Home Assistant Community Store](HACS.md) First 
Go to HACS in your sidebar.  
Go to Integrations  
Search for Tapo: Cameras Control  
Download/Install the Integration  
Restart HA  

## Other Dependencies  
Install [Frigate](../ADDONS/FRIGATE.md) and its dependencies
